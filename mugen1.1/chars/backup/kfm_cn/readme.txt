操作
========

基本操作
--------------

功夫人使用四个攻击键。X按键是轻拳，Y按键是重拳，A按键是轻脚，B按键是重脚。

以下是按键的表示方法：
D - 下
F - 前
U - 上
B - 后
x - X按键 (轻拳)
y - Y按键 (重拳)
a - A按键 (轻脚)
b - B按键 (重脚)

DF 表示下前，UB 后斜上，。ab代表A和B同时按。x/y代表按X或者Y键。

用方向键控制功夫人。按下方向键来蹲，上方向键来跳。在空中再按上方向键可以二连跳。按住与对手方位相反的方向键来防御。功夫人可以在空中防御。

当在空中被打中时，同时按X和Y按键来受身。受身时同时按方向键能向那个方向移动一段距离。当接近地面受身时，人物会做地面受身。

按start按钮可以让功夫人挑拨。


移动操作
-----------------

U              - 跳
UF             - 前跳
F              - 向前走
DF             - 蹲
D              - 蹲
DB             - 蹲或者蹲防
B              - 向后走或者站防
UB             - 后跳
F，F           - 前跑
B，B           - 后方躲闪
xy             - 受身

站立普通技
----------------------

x              - 轻拳
y              - 重拳
a              - 轻脚
b              - 重脚

蹲普通技
-----------------------

x              - 蹲轻拳
y              - 蹲重拳
a              - 蹲轻脚
b              - 蹲重脚

空中普通技
-----------------

x              - 跳轻拳
y              - 跳重拳
a              - 跳轻脚
b              - 跳重脚

投技
------

F/B + y (近距离) - 功夫投

格挡
------
F + x		- 上段格挡
DF + x		- 下段格挡
F + x (空中)	- 空中格挡

必杀技
---------------

D，DF，F，x    - 轻功夫掌
D，DF，F，y    - 重功夫掌
D，DF，F，x+y  - 疾风功夫掌 (消耗三分之一格能量)

F，D，DF，x    - 轻功夫升龙
F，D，DF，y    - 重功夫升龙
F，D，DF，x+y  - 疾风功夫升龙 (消耗三分之一格能量)

D，DB，B，x    - 轻功夫破
D，DB，B，y    - 重功夫破
D，DB，B，x+y  - 疾风功夫破 (消耗三分之一格能量)

D，DF，F，a    - 轻功夫铁山靠
D，DF，F，b    - 重功夫铁山靠
D，DF，F，a+b  - 疾风功夫铁山靠 (消耗三分之一格能量)

F，F，a        - 轻功夫腿
F，F，b        - 重功夫腿
F，F，a+b      - 疾风功夫腿 (消耗三分之一格能量)
在功夫腿中按A或B按键来踢第二下。

超必杀技 (消耗一格能量)
-------------

D，DF，F，D，DF，F，x/y  - 三连功夫掌
D，DB，B，D，DB，B，x/y  - 破裂功夫升龙


连招说明
===========

功夫人的长处在于能够用招式互相取消。以下是一些简单的取消规则:

1。 重普通技可以取消轻普通技。
    举例，轻脚 -> 重脚

2。 必杀技可以取消大部分普通技。
    举例，站重拳 -> 功夫掌

3。 超必杀技可以取消所有普通技和必杀技。
    举例，功夫掌 -> 三连功夫掌

4。 超必杀技可以取消其他必杀技。
    举例，三连功夫掌 -> 破裂功夫升龙

可以取消技能的能力使得功夫人可以打出很大伤害的连段。
